# Skill Fantasy

Open Source monster tamer game with machines and vehicles.

## Devlog Playlist

https://youtube.com/playlist?list=PLJsfx7Z0sXi7FZyrqkoQaW0OruHpnjxo-

## Build

Requirements:
+ xmake
+ c compiler (clang or gcc)
+ x11 (linux) or win32 (windows)

use xmake.

```sh
xmake build
xmake run
```
